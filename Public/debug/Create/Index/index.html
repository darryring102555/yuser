欢迎使用代码生成工具

选项 值 说明
模块名 必填:用于代码生成路径
控制器名 必填:用于代码生成路径及Controller文件;
方法名 必填:用于代码生成路径及生成前端js,文件等
数据表名 必填:用于获取数据表信息(不需要前缀)
title 选填:页面的二级title
是否必须登录才能访问:  是         否 选填:添加,修改数据时验证数据
是否创建数据MODEL类:  是         否 选填:选择是否生成Model类
是否创建前端model处理数据:  是         否 选填:生成前端model在前端处理数据
是否前端验证字段:  是         否 选填:添加或修改数据时前端验证字段
